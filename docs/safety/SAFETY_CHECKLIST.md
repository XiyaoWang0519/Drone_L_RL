## PRE-FLIGHT SAFETY CHECKLIST

### Personnel Safety
- [ ] All personnel completed safety training
- [ ] Roles and responsibilities assigned
- [ ] Emergency procedures briefed
- [ ] First aid kit available and accessible
- [ ] Emergency contact numbers posted
- [ ] All personnel wearing required PPE:
  - [ ] Safety glasses/goggles
  - [ ] Closed-toe shoes
  - [ ] Appropriate clothing (no loose items)

### Environment Assessment
- [ ] Flight area adequate size (min 6m x 6m x 4m)
- [ ] Flight zone clear of obstacles and personnel
- [ ] Safety zones marked and understood
- [ ] Emergency landing area designated
- [ ] Adequate lighting conditions (>500 lux)
- [ ] Environmental conditions acceptable:
  - [ ] Temperature: 15-30°C
  - [ ] Humidity: <70%
  - [ ] Wind: <15 mph (indoor <5 mph)
  - [ ] No precipitation

### Safety Equipment
- [ ] Class D fire extinguisher available
- [ ] LiPo safety bag/container present
- [ ] Sand bucket for battery emergencies
- [ ] Emergency stop remote functional
- [ ] First aid kit stocked and accessible
- [ ] Emergency lighting available
- [ ] Communication devices charged

### System Inspection
- [ ] Drone visual inspection completed (no damage)
- [ ] Propellers secure and undamaged
- [ ] All connections tight and secure
- [ ] Battery charge level >80%
- [ ] Battery temperature normal (<40°C)
- [ ] Battery condition good (no swelling/damage)
- [ ] Emergency stop system tested
- [ ] Communication links verified
- [ ] All sensors calibrated and functional

### Pre-Flight Testing
- [ ] Motor startup test completed
- [ ] Control response verified
- [ ] Emergency stop tested and functional
- [ ] Communication range tested
- [ ] UWB positioning system operational
- [ ] Failsafe systems verified
- [ ] Backup systems tested

### Documentation
- [ ] Flight plan reviewed and approved
- [ ] Risk assessment completed
- [ ] Insurance verification current
- [ ] Regulatory compliance verified
- [ ] Equipment certifications current

## FLIGHT OPERATIONS CHECKLIST

### Startup Sequence
- [ ] Final area clear announced
- [ ] All personnel in designated safe areas
- [ ] Ground station powered and operational
- [ ] UWB anchor system initialized
- [ ] Drone systems powered in sequence
- [ ] Final system status check "GO"
- [ ] Emergency stop armed and ready

### Takeoff
- [ ] "PREPARING FOR TAKEOFF" announced
- [ ] Final personnel position check
- [ ] Motors armed with confirmation
- [ ] Control response verified
- [ ] "TAKEOFF INITIATED" announced
- [ ] Hover established and stable
- [ ] All systems nominal

### During Flight
- [ ] Continuous visual contact maintained
- [ ] Safety observer monitoring continuously
- [ ] Emergency stop within immediate reach
- [ ] System parameters monitored
- [ ] Flight envelope respected
- [ ] Communication maintained
- [ ] Regular position announcements

### Landing
- [ ] "INITIATING LANDING" announced
- [ ] Landing area clear and confirmed
- [ ] Descent controlled and stable
- [ ] "LANDING COMPLETE" announced
- [ ] Motors disarmed immediately
- [ ] System powered down safely

---

## POST-FLIGHT SAFETY CHECKLIST

### Immediate Post-Flight
- [ ] All motors stopped and disarmed
- [ ] Battery disconnected safely
- [ ] System temperature check (normal)
- [ ] Personnel accounted for
- [ ] Area secured and safe

### Equipment Inspection
- [ ] Visual inspection for damage
- [ ] Propeller condition checked
- [ ] Motor temperature normal
- [ ] Battery condition assessed
- [ ] All connections verified secure
- [ ] Any damage documented

### Data and Documentation
- [ ] Flight logs downloaded
- [ ] Performance data recorded
- [ ] Any anomalies documented
- [ ] Test results recorded
- [ ] Maintenance items noted
- [ ] Safety checklist filed

---

## EMERGENCY RESPONSE CHECKLIST

### Emergency Stop Activation
- [ ] Emergency stop activated immediately
- [ ] "EMERGENCY STOP" announced loudly
- [ ] Area cleared of all personnel
- [ ] Situation assessed from safe distance
- [ ] Emergency services called if needed
- [ ] Incident documented thoroughly

### Battery Emergency
- [ ] Area evacuated immediately (10m minimum)
- [ ] Emergency services called (911)
- [ ] Fire suppression ready (Class D only)
- [ ] Area ventilated after incident
- [ ] Medical attention sought if exposure
- [ ] Incident reported and investigated

### Medical Emergency
- [ ] Scene made safe immediately
- [ ] First aid administered if qualified
- [ ] Emergency services called (911)
- [ ] Patient kept comfortable and monitored
- [ ] Incident documented and reported

### Key Safety Principles

- **WHEN IN DOUBT, DON'T** - Better to be conservative
- **COMMUNICATE** - Keep everyone informed
- **STAY ALERT** - Maintain situational awareness
- **BE PREPARED** - Know emergency procedures
